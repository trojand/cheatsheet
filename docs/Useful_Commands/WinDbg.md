# WinDbg

## Basic commands/shortcuts
++f6++ : Attach a process

++g++  : Go/Continue from breakpoint

++u++ : Unassemble (View/Display the assembly translation from memory)
`u` : Unassemble (View/Display the assembly translation from memory)
    * `u` : Display from EIP
    * `u <address>`
    * `u <symbol>`

`.reload /f` : reload modules 
`.cls` : clear screen
